## 🚀 About the Project

This is a full-stack Django To-Do application that allows users to manage daily tasks with authentication and complete CRUD functionality.

---

## ✨ Features

- User registration
- User login
- Create tasks
- View tasks
- Update tasks
- Delete tasks
- Mark tasks as completed
- Responsive design
- Dark mode support
